# Localização em tempo real

Esse simples projeto exibe, em um mapa, a localização do usuário em tempo real.

## Feito com

- Typescript
- React Native

## Bibliotecas

- expo
- expo-location
- react-native-maps

### Inicialização

```
npm install

npm start
```

Projeto criado com base em uma video aula da rocketseat.
